Connexion officier

Cet espace permet de gérer le calendrier de la guilde. Il faut une clé d'accès personnelle (jeton GitHub).

Première fois ? Crée ta clé (2 min) :
  1. Ouvre github.com → Fine-grained token (connecte-toi au compte qui a accès au repo).
  2. Repository access → Only select repositories → choisis La-Clauderie.
  3. Permissions → Repository permissions → Contents → Read and write.
  4. Génère le jeton, copie-le et colle-le ci-dessus.
