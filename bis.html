Guide de classe · Patch v0.28.0

Builds

Choisis ta classe, ton rôle — et ta spécialisation quand la classe en a plusieurs pour ce rôle : les talents (avec le pourquoi), la rotation, puis le meilleur équipement du jeu — bonus de sets compris, 2 anneaux inclus. Les 27 spés sont couvertes. Clique sur un objet, un talent ou un sort pour ouvrir sa fiche sans quitter la page.

Panoplies optimisées sur les données réelles du jeu (tag v0.28.0) avec les formules de combat du patch : la valeur de chaque objet est mesurée dans la panoplie complète, bonus de sets et procs compris — c'est pour ça que certaines pièces de set battent des objets « meilleurs » pris isolément. Alt. = meilleur remplaçant si la pièce te manque. équipable* = le tooltip cite d'autres classes, mais le jeu ne restreint les armures que par type (tissu/cuir/mailles). Le jeu n'a pas d'objets « uniques » : porter deux exemplaires du même anneau est autorisé (vérifié dans le code d'équipement) — d'où les anneaux doublés. FURY = acheté avec l'Honor PvP (v0.25) : son Warfare ne sert à rien en PvE, mais les stats brutes gagnent parfois quand même l'emplacement.
