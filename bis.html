Le meilleur stuff (BiS)

Choisis ta classe puis ton rôle : la liste montre la meilleure panoplie complète du jeu — bonus de sets compris (2, 3 et 4 pièces) — sur les 11 emplacements, 2 anneaux inclus.

Panoplies optimisées sur les données réelles du jeu (tag v0.23.0) avec les formules de combat du patch : la valeur de chaque objet est mesurée dans la panoplie complète, bonus de sets et procs compris — c'est pour ça que certaines pièces de set battent des objets « meilleurs » pris isolément. Alt. = meilleur remplaçant si la pièce te manque. équipable* = le tooltip cite d'autres classes, mais le jeu ne restreint les armures que par type (tissu/cuir/mailles).
